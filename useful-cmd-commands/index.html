Useful CMD commands

Jonathan Ródenas López

From time to time, it is useful to know some CMD commands. Here are the ones I consider most useful.

slmgr /upk Deactivate W10
tracert Displays the path followed by the packet.
cipher /W:C: It writes to all free space marked on disk C. This way, if you have formatted C, there is no way to recover any data. It doesn't have to be C, it can be any disk. You can also encrypt the files.
fc Compare files, just as diff does.
sfc /scannow Check and repair system files.
DISM /Online /Cleanup-Image /RestoreHealth It does a system restauration.
chkdsk Check disk status.
taskkill /f /t /Kill a task.
powercfg /energy Show energy info.
powercfg /batteryreport Show battery info.
netsh wlan show wlanreport Check wifi status.
nslookup Check DNS info of a domain name.
